# README — data/

Dossier de structuration des données : organisation par catégories et sous-thèmes.

Équipe assignée: Structuration des Données

Consignes principales:
- Rangez chaque CSV dans le dossier correspondant à sa **catégorie** et **sous-thème**.
- Exemple : `data/csv/Programmes_Academiques/Presentation_des_filieres/example.csv`.
- Documentez la source dans la colonne `Source` du CSV.
- Nommez les fichiers de façon claire : `<categorie>__<soustheme>__session_<X>.csv`.
- Tests et nettoyages → sur branches locales; ouvrez MR pour merge.
