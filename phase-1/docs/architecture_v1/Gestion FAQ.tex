% Section 1 : Introduction
\section{Introduction}
\label{sec:introduction}

Ce document présente l'architecture complète du \textbf{Système de Gestion FAQ ChatBot}. L'objectif principal est d'automatiser la collecte, la validation et l'insertion des questions/réponses depuis un formulaire Google Sheets vers une base de données SQLite locale, afin d'alimenter un chatbot conversationnel.

\subsection{Contexte et Objectifs}
\begin{itemize}
    \item \textbf{Automatiser} le processus de collecte des données depuis Google Sheets
    \item \textbf{Standardiser} le workflow de validation des données par l'équipe
    \item \textbf{Sécuriser} le stockage des données FAQ dans une base locale
    \item \textbf{Tracer} l'ensemble des opérations via des logs et archives
    \item \textbf{Documenter} l'architecture pour faciliter la maintenance
\end{itemize}

\subsection{Architecture Globale}
Le système suit une architecture en pipeline avec les étapes suivantes :

\begin{figure}[H]
    \centering
    \begin{minipage}{0.9\textwidth}
\begin{lstlisting}

        ↓               ↓           ↓               ↓
   Formulaire     sync.py    Traitement     import_validated.py
     Source      (export)     Manuel          (insertion BD)

\subsection{sync.py - Collecte des Données}
\label{subsec:sync-script}

Ce script est responsable de la collecte automatique des données depuis Google Sheets. Il s'exécute périodiquement et suit le processus suivant :

\begin{enumerate}
    \item Vérification de la connectivité internet
    \item Authentification avec l'API Google Sheets
    \item Récupération des \textbf{nouvelles données uniquement} (via suivi d'index)
    \item Export au format CSV horodaté
    \item Mise à jour de l'index de la dernière ligne traitée
    \item Journalisation des opérations
\end{enumerate}

\begin{lstlisting}[language=Python, caption=sync.py - Fonction principale]
def main():
    # Vérification de la connexion
    if not check_connection():
        log_status("FAILURE - No internet connection")
        return
    
    try:
        # Configuration des scopes API
        scope = [
            "https://spreadsheets.google.com/feeds",
            "https://www.googleapis.com/auth/drive"
        ]
        
        # Chargement des credentials
        key_file = Path(__file__).parent.parent / "config" / "faq-service-key.json"
        creds = ServiceAccountCredentials.from_json_keyfile_name(str(key_file), scope)
        client = gspread.authorize(creds)
        
        # Accès à la feuille
        sheet = client.open("FAQ ChatBot (réponses)").sheet1
        rows = sheet.get_all_records()
        
        # Création du fichier CSV horodaté
        timestamp = datetime.now().strftime("%Y-%m-%d_%H-%M")
        filename = f"new_data_{timestamp}.csv"
        filepath = Path("../data/pending") / filename
        
        # Export des données
        with filepath.open("w", newline="", encoding="utf-8") as csvfile:
            if rows:
                fieldnames = list(rows[0].keys())
                writer = csv.DictWriter(csvfile, fieldnames=fieldnames)
                writer.writeheader()
                writer.writerows(rows)
        
        log_status(f"SUCCESS - Fichier créé: {filename}")
        
    except Exception as e:
        log_status(f"FAILURE - {str(e)}")
        raise
\end{lstlisting}

\subsection{reset\_and\_import.py - Reset et Réimport}
\label{subsec:reset-script}

Ce script permet de recommencer le processus d'importation à zéro. Il est utile pour les tests ou en cas de besoin de revalidation complète.

\begin{lstlisting}[language=Python, caption=reset\_and\_import.py - Fonction principale]
def reset_and_import():
    # Suppression de la base de données
    db_path = Path("db/faq.db")
    if db_path.exists():
        db_path.unlink()
    
    # Déplacement des archives vers validated
    archived_dir = Path("../data/archived")
    validated_dir = Path("../data/validated")
    for file in archived_dir.glob("*.csv"):
        shutil.move(str(file), str(validated_dir / file.name))
    
    # Exécution de l'import
    subprocess.run([sys.executable, "import_validated.py"], cwd="."))
\end{lstlisting}

\subsection{Étapes Détaillées}

\textbf{1. Collecte Automatique}
\begin{itemize}
    \item Exécution périodique de \texttt{sync.py}
    \item Création de fichiers au format : \texttt{new\_data\_YYYY-MM-DD\_HH-MM.csv}
    \item Stockage initial dans \texttt{data/pending/}
\end{itemize}

\textbf{2. Validation Manuelle}
\begin{itemize}
    \item Déplacement des fichiers vers \texttt{data/processing/}
    \item Vérification et correction du contenu
    \item Validation finale et déplacement vers \texttt{data/validated/}
\end{itemize}

\textbf{3. Insertion en Base}
\begin{itemize}
    \item Exécution de \texttt{import\_validated.py}
    \item Lecture de tous les fichiers CSV dans \texttt{data/validated/}
    \item Insertion dans la table \texttt{faq} de \texttt{faq.db}
    \item Archivage automatique dans \texttt{data/archived/}
\end{itemize}

    \end{itemize}

\subsection{Compte de Service}
\begin{info}[Compte de Service]
Le compte de service permet une authentification sécurisée sans intervention utilisateur. Il est essentiel pour les processus automatisés.
\end{info}

\textbf{Configuration :}
\begin{enumerate}
    \item Navigation : \textbf{IAM \& Admin \to Service Accounts}
    \item Créer un compte : \texttt{faq-service}
    \item Attribuer le rôle : \textbf{Editor} ou \textbf{Sheets Editor}
    \item Générer une clé JSON : \texttt{faq-service-key.json}
\end{enumerate}

\begin{important}[Sécurité des Clés]
\begin{itemize}
    \item La clé JSON contient des identifiants sensibles
    \item Ne jamais commiter dans le dépôt Git
    \item Ajouter au \texttt{.gitignore} : \texttt{config/faq-service-key.json}
    \item Stocker dans un endroit sécurisé hors du projet
\end{itemize}
\end{important}

\subsection{Partage de la Feuille Google}
\begin{enumerate}
    \item Ouvrir la feuille : \texttt{FAQ ChatBot (réponses)}
    \item Cliquer sur \textbf{Partager}
    \item Ajouter l'email du compte de service :
    \begin{verbatim}
    faq-service@faq-chatbot-2026.iam.gserviceaccount.com
    \end{verbatim}
    \item Attribuer les permissions : \textbf{Reader} ou \textbf{Editor}
\end{enumerate}
